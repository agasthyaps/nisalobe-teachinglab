# **Things to Remember**

In order for Teaching Lab to successfully extract data from these CAPs, please keep the following in mind:

* Name the file as outlined [**above**]().   
* Name each tab as you see modeled in the template. **Cycle 1, Cycle 2**, etc. Feel free to add other tabs as resources \- they will not register when we analyze data.   
* For **group coaching**,   
  * Coaches should continue to create one CAP for the full group of teachers they support. To ensure all teachers are properly linked to that CAP on the backend, please follow the new [Group Coaching Rostering and CAPs Process](https://docs.google.com/document/d/1_7cnaHBnYBnan3wKCcvQJcg6CLbDWAygaK0NjziQ_ws/edit?tab=t.0) outlined by our L\&R team.  
  * For group coaching, please record average data across classrooms for the metrics section.  
* Please do not change the layout of this sheet (additional columns or tabs). If so, our CAP dashboard will be looking for information in the ***wrong*** places.   
* This CAP plan is meant to be used for either **one teacher**, or **one group of teachers**, across all of their cycles. Teachers with different goals should have different CAPs. 

All projects that have teacher or leader coaching, regardless of the cadence of coaching, should use the CAP. Coaches should work with their Coach Developer to determine the ideal number of coaching cycles for the project given the frequency of coaching.
